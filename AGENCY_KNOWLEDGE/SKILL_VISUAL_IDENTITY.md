# 🎨 SKILL: Identidade Visual & Branding Premium (Paulo)

Esta skill foca na extração e aplicação de identidades visuais de alta autoridade, especialmente para profissionais liberais e saúde.

## 🛠 Ferramentas de Extração
Para manter a fidelidade à marca, utilizamos scripts de análise cromática nos logos originais:
- **`scratch/get_rgb.py`**: Analisa os pixels dominantes.
- **`scratch/extract_logo.py`**: Isola elementos gráficos.
- **`scratch/make_black_logo.py`**: Gera variantes para backgrounds claros.

## 🎨 Paleta Hellen Trento (Referência)
- **Primary:** `#B9975B` (Dourado Champagne - Autoridade e Elegância)
- **Background Deep:** `#1A1A1A` (Cinza Antracita - Modernidade e Foco)
- **Accents:** `#FFFFFF` e variantes de opacidade para textos secundários.

## 📏 Princípios de Design
1. **Contraste Suave:** Evitar branco puro em fundo preto puro; usar cinzas muito escuros para reduzir a fadiga ocular.
2. **Tipografia:** Uso de fontes com peso variado para criar hierarquia clara (Ex: *Inter* ou *Outfit*).
3. **Espaçamento (Whitespace):** Manter respiro generoso entre seções para transmitir calma e organização.
