# 💻 SKILL: Componentes Front-End de Alta Conversão (João)

Padrões técnicos e estéticos para implementação de interfaces modernas com React e Vanilla CSS.

## 🏗 Stack Base
- **React + Vite:** Performance e DX superior.
- **Vanilla CSS:** Controle total sobre animações e especificidade (evitar frameworks pesados que engessam o design).

## 🧩 Componentes "Célebres" (Patterns)
1. **Interactive Tabs (Services):** Usar o estado do React para trocar conteúdos de forma fluida, sem reload.
2. **Step-by-Step (Process):** Visualização clara da jornada do cliente.
3. **Responsive Glassmorphism:** Uso de `backdrop-filter: blur()` e bordas sutis com baixa opacidade para um feeling premium.

## ⚡ Performance & Animações
- **Micro-interações:** Hovers suaves (`transition: all 0.3s ease`).
- **Lazy Loading:** Essencial para imagens pesadas (fotos da Hellen).
- **Icons:** Uso de SVG inline ou Phosphor Icons para leveza.
