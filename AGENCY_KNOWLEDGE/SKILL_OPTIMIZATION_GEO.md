# 🔍 SKILL: Otimização Multi-Motor - SEO, AEO, GEO & LLMO (Sobrau)

Estratégias para garantir que a marca apareça onde quer que o usuário busque (Google, Assistentes de Voz ou IAs Generativas).

## 📊 Níveis de Otimização
1. **SEO (Search Engine):** Estrutura `<h1>` a `<h6>`, alt tags em imagens, meta descriptions e títulos otimizados para "Psicóloga em [Cidade]".
2. **AEO (Answer Engine):** Uso de FAQs com estruturas diretas (Pergunta/Resposta) para capturar o "Featured Snippet" do Google e respostas da Alexa/Siri.
3. **GEO (Generative Engine):** Conteúdo semântico denso que fornece contexto para o Google SGE e Perplexity.
4. **LLMO (Large Language Model):** Criação de páginas de autoridade (Sobre, Bio, Artigos) que sirvam de base de fato para LLMs (ChatGPT, Claude) citarem a Hellen.

## 🛠 Checklist de Implementação
- [ ] JSON-LD (Schema.org): Definir `LocalBusiness` e `Person`.
- [ ] Open Graph: Imagens de preview otimizadas para compartilhamento.
- [ ] Sitemap & Robots.txt: Configuração correta para indexação rápida.
- [ ] Semantic HTML: Uso rigoroso de `<section>`, `<article>`, `<footer>`.
