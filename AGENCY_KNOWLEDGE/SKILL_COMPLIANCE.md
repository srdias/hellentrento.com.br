# 👩‍⚖️ Skill: Compliance & Ética (Dra. Beatriz)

Diretrizes para garantir que toda a comunicação da Hellen Trento seja ética, segura e profissional.

## 📖 Regras de Ouro (CRP)
1. **Sem Promessa de Cura:** Nunca usar termos como "resolva agora", "curar", "solução definitiva". Use "manejo", "tratamento", "autonomia", "regulação".
2. **Depoimentos:** Psicólogos não devem usar depoimentos de pacientes para fins de propaganda. Focar em conteúdos educativos e benefícios da técnica (TCC).
3. **Exposição de Casos:** Nunca usar nomes ou dados que identifiquem pacientes. Relatos devem ser fictícios e baseados em padrões clínicos.
4. **Valor do Serviço:** Não pode haver anúncio de preços em materiais de marketing público. O foco deve ser o valor técnico.

## 🔒 LGPD para Psicologia
1. **Formulários:** Devem ter checkbox de consentimento claro.
2. **Dados Sensíveis:** Minimizar a coleta no site. Coleta de dados clínicos só no consultório.
3. **Base Legal:** O tratamento de dados no site deve ser baseado no "legítimo interesse" ou "consentimento".

## 📋 Checklist de Revisão (Gate 1)
- [ ] O texto contém promessas de resultado garantido?
- [ ] Existe exposição inadequada de terceiros?
- [ ] O tom é "mercantilista" demais ou soa como saúde profissional?
- [ ] A política de privacidade está acessível no footer?
